# YouTube-Video-Summarizer
<b>YouTube Video Summarizer Using Open Source LLMs </b>

This project is an intelligent YouTube Video Summarizer that extracts transcripts from videos and generates concise, meaningful summaries using advanced open-source Large Language Models (LLMs).

It also features browser access, allowing the system to fetch real-time, up-to-date information from the web, enhancing summaries with the latest context when needed.

<b> FEATURES </b>

<b> AUTOMATIC TRANSCRIPT EXTRACTION </b>

Fetches subtitles directly from YouTube videos

<b> AI-POWERED SUMMARIZATION  </b>

Converts long video content into clear, structured summaries

<b> BROWSER ACCESS (LIVE KNOWLEDGE RETRIEVAL) </b>

Enhances summaries with current information from the internet

<b> FAST & EFFICIENT PROCESSING </b>

Supports local or API-based LLM inference

<b> MODULAR DESIGN </b>

Easily extendable for different models or workflows
Change models as needed

<b> OPTIONAL UI SUPPORT </b>

Can be integrated with Streamlit or other frontends

<b> TECH STACK </b>

Python

YouTube Transcript API

LangChain (optional)

Open-Source LLMs (Ollama / Groq)

Web Browsing Tools

<b> USE CASES </b>

Quick understanding of long YouTube videos

Study notes generation

Content research & analysis

Productivity enhancement

Summarizing lectures, podcasts, and tutorials

<b> HOW IT WORKS </b>

User provides a YouTube video URL

Transcript is extracted using YouTube Transcript API

Text is cleaned and optionally punctuated

AI model generates a concise summary

(Optional) Browser retrieves latest context to enhance output
